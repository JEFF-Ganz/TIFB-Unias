Selamat datang di Admin Panel TIF-B CyberSquad.


Semua perubahan yang Anda buat di sini akan langsung terlihat di website utama secara realtime melalui Firebase Realtime Database.


  • Klik panel di sidebar untuk mulai mengedit
  • Tekan "Simpan" untuk menyimpan perubahan ke database
  • Website akan otomatis update tanpa perlu refresh
Inisialisasi Database

Jika database Firebase masih kosong (data anggota, jadwal, galeri tidak muncul di admin), klik tombol di bawah untuk mengisi database dengan semua data awal dari website.
⚠️ Data yang sudah ada di database tidak akan ditimpa.
